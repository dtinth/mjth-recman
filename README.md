This little service lets clients control multi-track recording in Jamulus via chat.

Service dependencies:

1. [Jamulus JSON-RPC API gateway](https://github.com/dtinth/jamulus-json-rpc-api-gateway)
2. [gojam](https://github.com/dtinth/gojam)
3. [upload-endpoint](https://github.com/dtinth/upload-endpoint)
